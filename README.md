# Productivity Hub
[Live Demo](https://harshitverma-06.github.io/Movie-Search-App/)

A simple and responsive web app to use productivity tools . 

## Features
- ToDo list
- Pomodoro Timer
- Habit Tracker
- Notes saver
- Daily Quotes
- Responsive design with Hamburger menu layout for small screens 
- Saves data to local storage

## Tech Stack
- HTML
- CSS
- JavaScript
